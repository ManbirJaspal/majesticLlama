How did I create this website?

I've used AngularJS to implement the Majestic Llama challenge. I used AngularJS because it was really easy to setup and setup.

The page routing is defined in config.js using the Angular's routing implementation. All the pages get loaded into the container div defined inside index.html

I've implemented the MVC architecture. I've separated views and their associated controllers.

I've taken help from online resources to build the game. I had never used a canvas before. So it was a bit of a learning curve towards understanding how a canvas works.

I made use of bootstrap to implement the nav bar and make it responsive. I've also used bootstrap classes for other DOM elements to make sure they are responsive.

I've hosted the website on an AWS S3 bucket.
